# 基础语法实验

目录中的短程序按主题保存。它们用于观察单个语言规则，不代表生产代码结构。

| 路径 | 内容 |
| --- | --- |
| `sequences` / `strings` | 数组、`vector` 与字符串 |
| `classes` | 类、构造与销售数据示例 |
| `inheritance` / `polymorphism` | 继承、访问控制和虚函数 |
| `types` | 初始化、指针和类型规则 |
| `templates` | 函数模板与类模板 |
| `polymorphism-shapes` | 运行时多态综合示例 |
| `object-model` | 多文件对象关系实验 |
| `containers` / `sorting` | 容器与算法实验 |

含“错误示例”的文件应保持默认可编译：错误语句放在注释中，紧邻解释失败原因。
